## Prerequisites

- macOS 14+ with Apple Silicon (arm64)
- [Pixi](https://pixi.sh) installed:

```bash
curl -fsSL https://pixi.sh/install.sh | bash
```

## Installation Options

### Option A — Pixi package (recommended for most users)

The compiled `Emacs.app` is published to the [jwintz Pixi channel](https://prefix.dev/channels/jwintz/packages/emacs) and can be installed without building from source:

```bash
pixi global install emacs --channel https://prefix.dev/channels/jwintz --channel conda-forge
```

This downloads a pre-built, codesigned `Emacs.app` with native compilation and all macOS patches already applied.

> [!tip] Use the package if you don't need to modify patches
> Building from source takes 15–30 minutes. The Pixi package is identical to a local build and is the fastest way to get started.

### Option B — Build from source

Build the full pipeline yourself to modify patches, test upstream changes, or produce a custom `Emacs.app`:

## Setup

```bash
git clone --recurse-submodules <repository-url>
cd hyalo-feedstock
pixi install
```

The `emacs/` submodule is the pristine Emacs source — it is never modified.

## Build

Run the full pipeline:

```bash
pixi run prep       # Sync emacs/ → emacs-build-macos/
pixi run patch      # Apply macOS patches
pixi run autogen    # ./autogen.sh
pixi run configure  # ./configure --with-ns --with-modules --with-native-compilation ...
pixi run build      # make -j$(nproc)
pixi run install    # Create and codesign Emacs.app
```

Or launch directly:

```bash
pixi run run        # build → install → launch Emacs.app
```

## Reset

If the build directory becomes inconsistent:

```bash
pixi run clean-builds   # Remove emacs-build-macos/
pixi run prep           # Recreate from pristine source
```
