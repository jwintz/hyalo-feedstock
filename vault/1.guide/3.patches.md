Patches live in `patches/` and are applied by `pixi run patch` to `emacs-build-macos/`.

The pristine `emacs/` submodule is never touched — all patches apply to the ephemeral build directory.

## Patch Categories

### Transparency

Enables per-frame background transparency for the Emacs.app window. Used by Hyalo to set the window's background alpha from Swift.

### Appearance

Hooks into macOS appearance changes (light/dark mode switching) so Emacs Lisp can react to system appearance. Hyalo uses this to synchronize theme loading with the system appearance.

### Fixes

macOS-specific bug fixes and compatibility patches that haven't landed upstream or need backporting.

## Adding a Patch

1. Make changes in `emacs-build-macos/`
2. Generate a patch:
   ```bash
   diff -u emacs/<file> emacs-build-macos/<file> > patches/my-patch.patch
   ```
3. Commit the patch to `patches/`
4. Run `pixi run prep && pixi run patch` to verify it applies cleanly

## Resetting Patches

```bash
pixi run clean-builds && pixi run prep && pixi run patch
```
